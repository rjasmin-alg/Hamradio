## 游섫릖 English

Thank you for your interest in contributing to the Ham Radio Handbook! Your help is essential for maintaining the accuracy and quality of the information.

### How can you help?
*   **Bug fixes**: If you spot a typo or incorrect information, feel free to fix it.
*   **Adding content**: Expert in a specific topic? Write a new article or expand an existing one.
*   **Translations**: Help us translate content into English.

### Process
1.  **Fork the repository**: Create your copy of the project on GitHub.
2.  **Create a branch**: `git checkout -b feature/change-name`
3.  **Make changes**: Edit Markdown files in the `docs/` folder.
4.  **Test locally**: Run `mkdocs serve` and check the preview.
5.  **Submit a Pull Request**: Describe what you changed and why.

### Writing Style
*   Use clear and understandable language.
*   Format text using standard Markdown.
*   Verify the accuracy of technical information.
