## üá¨üáß English

Welcome to the **Ham Radio Handbook**, a comprehensive resource for ham radio operators, from beginners to experts. This project aims to gather all relevant information about amateur radio in one place.

### Content
The project covers a wide range of topics:
*   **Regulations & Licensing**: How to become a ham, laws, regulations, and callsigns.
*   **Technical**: Electronics basics, antennas, transceivers, safety, and digital networks (Hamnet).
*   **Operating**: Band plans, modes, procedures, and logging.
*   **CB Radio**: Everything about Citizens Band radio.
*   **Activities**: Contesting, DXing, portable operations, satellites, and ARDF.
*   **Emergency Communications**: Ham radio in emergencies (EmComm, Winlink).
*   **Community**: Associations, clubs, and youth activities (YOTA).

### How to run locally
This documentation is built using [MkDocs](https://www.mkdocs.org/) and [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).

1.  **Install Python**: Ensure you have Python installed.
2.  **Install dependencies**:
    ```bash
    pip install mkdocs-material
    pip install mkdocs-i18n  # If needed for i18n
    ```
3.  **Start the server**:
    ```bash
    mkdocs serve
    ```
4.  Open your browser at `http://127.0.0.1:8000/`.
