# Frekvencijski planovi

Frekvencijski plan (Band Plan) je dogovor o korištenju dijelova amaterskih opsega za različite vrste rada. Poštivanje band plana je stvar radioamaterske etike i kulture.

Ovdje prikazujemo plan za **IARU Regiju 1 (Europa)**, uz napomene za **Regiju 2 (Sjeverna i Južna Amerika)** gdje postoje značajne razlike.

## HF (Kratki val)

### 160m (1.8 MHz)
"Top Band". Noćni opseg.

| Frekvencija (kHz) | Namjena (EU) | Napomena (USA) |
| :--- | :--- | :--- |
| 1810 - 1838 | CW | USA: 1800 - 2000 kHz |
| 1838 - 1840 | Digital (FT8 @ 1840) | |
| 1840 - 2000 | SSB / CW | |
