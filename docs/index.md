# Dobrodošli u Radioamaterski Priručnik

Ovo je centralno mjesto za sve informacije o radioamaterizmu. Bilo da ste početnik koji želi položiti ispit ili iskusni operator koji traži specifične tehničke informacije, ovdje ćete pronaći sve što vam treba.

## Što je radioamaterizam?

Radioamaterizam je hobi koji spaja ljude diljem svijeta putem radio valova. To je tehnički hobi koji uključuje konstruiranje, eksperimentiranje i komunikaciju.
