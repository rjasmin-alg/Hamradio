# Software for Radio Amateurs

## Introduction
In modern amateur radio, the computer has become an inseparable part of the amateur radio station (shack). From logging contacts and managing digital modes to Software Defined Radios (SDR), software is just as critical as the transceiver or antenna itself.

## How to use this list
This comprehensive list covers numerous tools, but we recommend focusing on the basics first. Here is the "Starter Pack":
- **WSJT-X**: The absolute standard for FT8/FT4 digital modes.
- **Log4OM v2**: A highly capable and free Windows application with excellent integration, arguably the best general logging tool.
- **MMANA-GAL**: One of the best beginner tools for designing and modeling shortwave wire antennas.
- **HamAlert**: A crucial mobile tool for tracking propagation and spots.
- **LCWO.net**: The best web tool for mastering telegraphy (CW).

!!! tip "Recommended list for beginners"
    By downloading this "Starter pack" you will cover most of the requirements to start operating on frequencies, maintain a logbook, and test your first digital modes. Coupled with mastering telegraphy via *LCWO*, you have a great start!

## Logging
The logbook is the central software of every station, ensuring the safe and organized storage of contacts.
* **N1MM+**: Leading world logger for contesting. (Win / Free)
* **Log4OM v2**: Complete and modern general logger. (Win / Free)
* **CloudLog**: Excellent web-based self-hosted logger. (Web / Free)
* **Ham Radio Deluxe**: "All-in-one" system for rig control and logging. (Win / Paid)

## Digital Modes
Digital modes have changed HF communications, enabling stable links even in poor propagation.
* **WSJT-X**: Dominant tool for FT8, FT4, WSPR. (Win/Mac/Linux / Free)
* **fldigi**: Foundation software for legacy modes like PSK31, RTTY. (Win/Mac/Linux / Free)
* **JS8Call**: Allows free text utilizing the robust FT8 routing. (Win/Mac/Linux / Free)
* **Winlink Express**: Standard for transferring e-mail messages via RF without internet. (Win / Free)

## SDR (Software Defined Radio)
Applications responsible for processing uncommonly large bands of RF signals.
* **SDR#**: Popular and versatile viewer for RTL-SDR dongles. (Win / Free)
* **SDRuno**: Main software for SDRPlay receivers. (Win / Free)
* **Gqrx**: GNU Radio-based application excelling on Linux. (Mac/Linux / Free)

## Antenna Modeling
Designing your own antennas and predicting their behavior.
* **MMANA-GAL**: Golden free standard for beginners. (Win / Free)
* **4NEC2**: Advanced NEC2-based simulator. (Win / Free)

## Propagation Tracking
Applications for tracking solar cycles and impacts on the ionosphere.
* **VOACAP**: Point-to-point prediction of stable radio communications. (Web / Free)
* **HamClock**: All-in-one display for tracking satellites, solar data, and spots. (Linux/Web/Pi / Free)
* **SolarHam**: Central site for evaluating solar flares and K-indices. (Web / Free)

## CW Practice (Learning Telegraphy)
* **LCWO.net**: Standard for online learning using the effective Koch Method. (Web / Free)
* **Just Learn Morse**: Desktop tool utilizing auditory templates. (Win / Free)

## Radio Control (CAT)
Intermediate library layers to communicate directly with hardware.
* **Hamlib**: Open-source foundation covering over 200 modern devices. (Win/Mac/Linux / Free)
* **OmniRig**: Highly popular Windows COM interface recognizing multiple VFOs. (Win / Free)

## Mobile Applications
* **HamAlert**: Notifications for spotting specific DX. (iOS/Android / Free)
* **RepeaterBook**: Offline repository of repeater locations globally. (iOS/Android / Free)
* **APRS.fi**: Visualizing user movements on the digital-location APRS system. (iOS/Android / Free)
* **SOTA Spotter**: Tracking mobile station activations on mountain summits. (iOS/Android)
