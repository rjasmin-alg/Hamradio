# Propisi i Kanali

Iako nema ispita, postoje pravila koja se moraju poštivati.

## Dozvoljeni parametri (Hrvatska / EU)
- **Frekvencijski opseg**: 26.965 MHz - 27.405 MHz.
- **Modulacija**: AM, FM, SSB (LSB/USB).
- **Snaga**: 4W (AM/FM), 12W (SSB).
