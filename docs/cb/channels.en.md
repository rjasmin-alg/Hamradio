# Regulations and Channels

Although there is no exam, there are rules that must be followed.

## Allowed Parameters (Croatia / EU)
- **Frequency Range**: 26.965 MHz - 27.405 MHz.
- **Modulation**: AM, FM, SSB (LSB/USB).
- **Power**: 4W (AM/FM), 12W (SSB).
