# Hamnet (High Speed Multimedia)

Hamnet je brza radioamaterska mreža bazirana na IP protokolu, koja koristi modificiranu WiFi opremu na radioamaterskim frekvencijama.

## Sadržaj

### Uvod i infrastruktura
Što je Hamnet, kako radi i koja se oprema koristi (Ubiquiti, Mikrotik).

### Njemačka iskustva i rješenja (DAeC)
Njemačka ima jednu od najrazvijenijih Hamnet mreža na svijetu. Ovdje ćemo prenijeti njihova iskustva i najbolje prakse.

### Primjena
- Povezivanje repetitora
- Video nadzor
- VoIP telefonija
- Pristup serverima i servisima unutar mreže
