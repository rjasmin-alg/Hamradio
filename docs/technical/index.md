# Tehnika

Tehnički aspekt je srž radioamaterizma. Ovdje obrađujemo sve od osnova elektronike do naprednih komunikacijskih sustava.

## Sadržaj

- **Osnove elektronike**: Otpornici, kondenzatori, zavojnice, poluvodiči.
- **Antene i vodovi**: Dipoli, Yagi, vertikalke, koaksijalni kabeli.
- **Primopredajnici i oprema**: Kako odabrati uređaj, dodatna oprema.
- **Sigurnost u radu**: Uzemljenje, zaštita od groma, RF zračenje.
- **Samogradnja (DIY)**: Projekti za samostalnu izradu.
- **[Hamnet](hamnet.md)**: High Speed Multimedia mreža.
