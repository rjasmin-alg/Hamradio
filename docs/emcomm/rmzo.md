# RMZO (Radioamaterska mreža za opasnost)

RMZO je sustav unutar Hrvatskog radioamaterskog saveza koji se aktivira u slučaju katastrofa.

## Organizacija
- **Nacionalna razina**: Koordinira HRS u suradnji s Ravnateljstvom civilne zaštite.
- **Lokalna razina**: Radioklubovi i pojedinci koji se stavljaju na raspolaganje lokalnim stožerima.

## Aktivacija
RMZO se aktivira na poziv nadležnih tijela (112) ili samoinicijativno u slučaju očitog prekida telekomunikacija.

## Frekvencije
- **3.675 MHz (80m)**: Glavna frekvencija za nacionalnu koordinaciju.
- **145.500 MHz (2m)**: Lokalna simpleks frekvencija.
- **Repetitori**: Koriste se lokalni repetitori koji imaju neovisno napajanje.
