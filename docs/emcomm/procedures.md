# Postupci u hitnim slučajevima

Kada se dogodi nesreća, disciplina je ključna.

## Zlatna pravila
1.  **Slušaj**: Ne odašilji ako nisi pozvan ili nemaš hitnu informaciju.
2.  **Kratko i jasno**: Koristi fonetsku abecedu. Izbjegavaj žargon.
3.  **Zapiši**: Sve poruke moraju biti zapisane (vrijeme, pošiljatelj, sadržaj).

## Vrste poruka
- **Mayday / SOS**: Neposredna životna opasnost. Apsolutni prioritet.
- **Pan-Pan**: Hitna situacija, ali nema neposredne opasnosti po život.
- **Securite**: Sigurnosno upozorenje (npr. o vremenu).

## IARU Emergency Message
Standardizirani format za slanje poruka koji osigurava da se ništa ne izgubi u prijenosu.
