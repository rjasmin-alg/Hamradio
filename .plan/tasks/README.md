---

## Workflow (s bilo kojeg računala)

```
1. git pull
2. Otvori task datoteku po izboru
3. Pročitaj checklist — vidi što nedostaje
4. Kopiraj prompt iz sekcije "Prompt"
5. Pokreni agenta s tim promptom
6. Agent zapisuje u docs/
7. Označi završene korake u checklisti
8. git add, git commit, git push
```
